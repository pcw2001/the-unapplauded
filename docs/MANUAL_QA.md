# The Unapplauded — 手动测试指南

版本：v0.2 local prototype

## 测试环境

```bash
npm install
npm run dev
```

打开 http://localhost:3000，用浏览器 DevTools 模拟移动端（375px）和桌面端（1280px）。

## 测试用例

### 1. 首页输入

| 步骤 | 预期结果 |
|------|---------|
| 打开首页 | 看到标题 "The Unapplauded"、副标题 "一个存放小事的地方"、输入框、按钮 |
| 不输入任何内容，直接点"生成展品" | 出现提示："写点什么吧，哪怕很小的事。" |
| 输入超过 200 个字 | 无法继续输入，字数显示 200/200 |
| 输入 "今天给自己煮了一碗面"，点"生成展品" | 跳转到预览页，看到展品 |
| 输入后按 Enter 键 | 和点击按钮效果相同 |
| 输入后按 Shift+Enter | 换行，不触发提交 |

### 2. 展品生成与预览

| 步骤 | 预期结果 |
|------|---------|
| 生成展品后 | 预览页显示展品编号、展厅名、标题、材质、展签、策展人笔记 |
| 点"换一种呈现" | 同一输入生成不同的标题/展签/笔记，编号不变 |
| 多次点"换一种呈现" | 每次随机组合，不会每次都一样 |
| 直接访问 /preview（无展品数据） | 自动跳转回首页 |

### 3. 保存到博物馆

| 步骤 | 预期结果 |
|------|---------|
| 点"放入博物馆" | 出现确认页："已放入博物馆"，有"回到大厅"和"查看博物馆"按钮 |
| 点"查看博物馆" | 跳转到博物馆页，看到刚保存的展品 |
| 点"回到大厅" | 跳转回首页 |

### 4. localStorage 持久化

| 步骤 | 预期结果 |
|------|---------|
| 保存展品后刷新页面 | 展品仍然存在 |
| 关闭浏览器标签页，重新打开 | 展品仍然存在 |
| 打开新的隐身窗口 | 看不到展品（不同浏览器实例） |

### 5. 博物馆空状态

| 步骤 | 预期结果 |

| 清除 localStorage 后访问 /museum | 看到空状态插图、"博物馆还没有展品"、"写下第一件小事"按钮 |

### 6. 博物馆墙（有展品）

| 步骤 | 预期结果 |
|------|---------|
| 有多件展品时 | 网格布局显示所有展品卡片 |

### 7. 展品详情弹窗

| 步骤 | 预期结果 |
|------|---------|

## 中文测试输入（20 条）

用于快速测试展厅匹配和生成质量。每条输入后标注了期望的展厅分类。

### 厨房 / 自我照料

1. `今天给自己煮了一碗面，加了一个鸡蛋` → 厨房厅
2. `洗了一个热水澡，洗了很久` → 自我照料厅
3. `热了昨天的剩饭，一个人吃完了` → 厨房厅
4. `喝了三杯水，终于不是渴着嗓子过完一天` → 自我照料厅

### 整理 / 秩序

5. `把桌子收拾干净了，终于能看到桌面` → 日常秩序厅
6. `叠了一堆衣服，放进衣柜里` → 日常秩序厅
7. `扫了地，地板上不再有碎屑` → 日常秩序厅
8. `把冰箱里的过期食物扔掉了` → 日常秩序厅

### 社交 / 关系

9. `给妈妈打了一个电话` → 社交勇气厅
10. `回了朋友三天前的消息` → 社交勇气厅
11. `和同事一起吃了午饭` → 社交勇气厅
12. `跟房东说了热水器的问题` → 社交勇气厅

### 工作 / 学习

13. `写完了拖了一周的报告` → 日常秩序厅（默认）
14. `看了二十页书` → 日常秩序厅（默认）
15. `整理了电脑桌面的文件` → 日常秩序厅

### 休息 / 低能量日

16. `躺了一下午什么也没做` → 休息厅
17. `午睡睡了两个小时` → 休息厅
18. `今天没有出门，但也没有自责` → 休息厅

### 情绪困难

19. `哭了一场，不知道为什么` → 情绪管理厅
20. `今天特别丧，但还是吃了晚饭` → 情绪管理厅

## 语气检查清单

生成 10 件展品后，逐条检查：

- [ ] 标题不像打卡记录（没有"第X天""连续""目标"）
- [ ] 标题不像励志语录（没有"加油""你可以""相信自己"）
- [ ] 展签有具体的画面感（有物件、有场景、有时间）
- [ ] 材质混合了具体物件和少量感受（不全是抽象名词）
- [ ] 策展人笔记像展签旁边的轻声说明，不像口号
- [ ] 不同展厅的语气有差异（厨房有烟火气，休息厅更慵懒）
- [ ] 同一展厅内多次生成不会感到重复

## 已知限制

- 数据仅存在于当前浏览器的 localStorage
- 清除浏览器缓存会丢失所有展品
- 不支持编辑已保存的展品
- 不支持导出展品
- 展品文案基于随机模板，不使用 AI
- 无深色模式
- 无跨设备同步
